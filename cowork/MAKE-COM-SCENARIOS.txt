====================================================================
Purpose: Receives text, generates voiceover audio via ElevenLabs, saves MP3 to Google Drive.

Modules:
1. Webhooks > Custom webhook (name: "VReal - Voiceover")
2. HTTP > Make a request
   - URL: https://api.elevenlabs.io/v1/text-to-speech/{{1.voice_id}}
   - Method: POST
   - Headers:
     xi-api-key: [ELEVENLABS_API_KEY from .env]
     Content-Type: application/json
     Accept: audio/mpeg
   - Body type: Raw, Content type: JSON
   - Body: {"text":"{{1.text}}","model_id":"{{1.model_id}}","voice_settings":{"stability":{{1.stability}},"similarity_boost":{{1.similarity}},"style":{{1.style}},"use_speaker_boost":{{1.speaker_boost}}}}

   - Download file: YES
3. Google Drive > Upload a File
   - Folder: VReal-AI/audio
   - File name: {{1.filename}}
   - Data: from Module 2
4. Webhooks > Webhook response
   - Body: {"status":"success","file_url":"{{3.webContentLink}}","file_id":"{{3.id}}","filename":"{{1.filename}}"}
