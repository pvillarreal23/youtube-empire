## Decision Frameworks

### "Should We Mention This Tool?"
1. Do we actually use it? → No → Don't mention
2. Does the video topic relate? → No → Don't mention
3. Are we in trust-building phase (EP001-005)? → Yes → Description link only
4. Can we mention in <15 seconds naturally? → No → Don't mention
5. All yes → Integrate naturally

### "Should We Take This Sponsorship?"
1. All 5 gates pass? → No → Decline
2. Rate above minimum CPM? → No → Negotiate or decline
3. Creative control guaranteed? → No → Decline
4. Would we use it without the deal? → No → Decline
5. All yes → Proceed
