# Vault Index

## Folders

- **raw/** — Incoming, unprocessed content. Drop new notes, clippings, and ideas here for later organization.
- **wiki/** — Organized, interlinked notes. The main knowledge base with wikilinks connecting related topics.
- **templates/** — Reusable note structures and templates for consistent formatting across the vault.
- **projects/** — Project-specific notes and resources.
  - **projects/v-real-ai/** — Notes and materials for the V-Real AI project.
  - **projects/personal/** — Personal project notes and references.
- **journal/** — Daily notes and reflections.
- **archive/** — Completed or inactive projects moved here for long-term storage.
