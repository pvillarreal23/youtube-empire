## How to Use This Vault

1. **Copy any prompt** and paste it into ChatGPT, Claude, or Gemini
2. **Replace the [brackets]** with your own details
3. **Don't overthink it** — the prompt does the heavy lifting, you just fill in the blanks
4. Every prompt has been tested. If it's in here, it works.

**Pro tip:** Start with Section 1 if you've never used AI before. Work your way down. By Section 8, you'll wonder how you ever worked without this.

## SECTION 1: Getting Started (Your First 10 Minutes with AI)

These prompts are for people who have never used AI or have only tried it once or twice. No jargon. Just results.

### 1.1 — Your First Real Conversation
```
I'm brand new to AI. I work as a [your job title] in [your industry].
I have about 30 minutes. Show me the 3 most useful things I can do
with you right now that will save me time this week. Keep it simple —
explain like I'm smart but have never done this before.
```

### 1.2 — Rewrite Anything Better
```
Rewrite this to be clearer, more professional, and half the length.
Keep my main point but make it sound like someone who knows what
they're talking about:

[paste your email, message, or document here]
```

### 1.3 — Explain Anything Like I'm New
```
Explain [topic] to me like I'm intelligent but have zero background
in this subject. Use one real-world analogy. Keep it under 200 words.
No jargon.
```

### 1.4 — Summarize Long Documents Instantly
```
Read this and give me:
1. A 3-sentence summary
2. The 5 most important points
3. Any action items or deadlines mentioned

[paste document, article, or email thread here]
```

### 1.5 — Make Decisions Faster
```
I'm trying to decide between [option A] and [option B].
Here's my situation: [brief context].
Give me a clear pros/cons comparison and tell me which one you'd
pick if you were in my shoes. Be direct.
```

### 1.6 — Turn Messy Notes into Organized Docs
```
Here are my rough notes from [a meeting / a brainstorm / my research].
Organize them into a clean, structured document with headers and
bullet points. Group related ideas together. Flag anything that
seems like an action item.

[paste messy notes here]
```

### 1.7 — Write an Email You've Been Putting Off
```
Write a professional email for this situation:
- I need to: [what you want to accomplish]
- I'm writing to: [their role/relationship to you]
- The tone should be: [friendly / formal / apologetic / firm]
- Key points to include: [list 2-3 things]

Keep it under 150 words. No fluff.
```

### 1.8 — Prepare for Any Meeting in 5 Minutes
```
I have a meeting in [time] about [topic] with [who].
My role is [your role]. Help me prepare:
1. 3 smart questions to ask
2. Key points I should bring up
3. Anything I should watch out for
4. A 30-second summary of my position if asked
```

### 1.9 — Learn a New Skill Fast
```
I want to learn [skill] well enough to [specific goal] within
[timeframe]. I'm starting from zero. Create a focused learning plan:
- What to learn first (and what to skip)
- The best free resources
- A daily practice routine (15-30 min max)
- How I'll know when I'm "good enough"
```

### 1.10 — Daily Productivity Kickstart
```
Here's my to-do list for today:
[paste your list]

Help me:
1. Prioritize these by actual impact (not urgency)
2. Identify which ones I can automate or delegate
3. Estimate realistic time for each
4. Tell me which one to start with and why
```

## SECTION 2: Writing & Content

### 2.1 — Blog Post from Scratch
```
Write a blog post about [topic].
Audience: [who reads this]
Tone: [casual / professional / educational]
Length: ~[number] words
Include: a compelling opening, 3-5 key points with examples,
and a conclusion with a clear takeaway.
Don't start with "In today's..." or "In the world of..."
```

### 2.2 — Social Media Post Generator
```
Write 5 social media posts about [topic] for [platform].
Each post should:
- Hook attention in the first line
- Deliver one clear insight
- End with engagement (question, CTA, or hot take)
- Be the right length for [platform]
Tone: [your brand voice]
```

### 2.3 — Turn a Long Piece into Short Content
```
Take this long-form content and create:
1. A Twitter/X thread (5-7 tweets)
2. A LinkedIn post (150 words)
3. An Instagram caption
4. A YouTube Shorts script (60 seconds)

Keep the core message but adapt the format and tone for each platform.

[paste your long-form content]
```

### 2.4 — Headline Generator
```
Generate 10 headline options for content about [topic].
Audience: [who]
Goal: [clicks / shares / sign-ups / purchases]
Mix styles: curiosity gap, how-to, list, contrarian, data-driven.
Star the 3 you think would perform best and say why.
```

### 2.5 — Script for YouTube / Podcast
```
Write a [length]-minute script about [topic].
Format: [documentary narration / conversational / educational]
Structure: Hook (15 sec) → Setup (1 min) → Core content →
Takeaway → CTA
Audience: [who, what they care about]
Voice style: [e.g., "BBC documentary narrator" or "smart friend
explaining over coffee"]
Include [PAUSE] markers where the speaker should breathe.
```

### 2.6 — Newsletter Email
```
Write a newsletter email about [topic/update].
Subject line options: give me 3
Preview text: write the first line that shows in inbox
Body: under 300 words, conversational, one key insight or story,
end with a clear CTA
Audience: [describe your subscribers]
```

### 2.7 — Fix My Writing
```
Review this writing and improve it. Don't rewrite it entirely —
preserve my voice. Just fix:
- Awkward phrasing
- Unnecessary words (cut the fluff)
- Weak verbs (make them stronger)
- Unclear sentences

[paste your draft]
```

## SECTION 3: Business & Career

### 3.1 — Business Idea Validator
```
I have a business idea: [describe it in 2-3 sentences].
Act as a skeptical but fair business advisor. Give me:
1. The strongest argument FOR this idea
2. The strongest argument AGAINST it
3. Who the ideal customer is (be specific)
4. What I should test first before investing money
5. An honest 1-10 rating on viability
```

### 3.2 — Competitive Analysis
```
I'm in the [industry] space. My main competitors are [list 2-3].
Analyze their strengths and weaknesses compared to what I offer:
[describe your product/service].
Where's the gap I can own? What are they doing that I should steal?
What are they doing wrong?
```

### 3.3 — Resume Bullet Points
```
Rewrite these job responsibilities as achievement-focused bullet
points for a resume. Use numbers and results wherever possible.
Make them sound impressive but honest.

Job: [title] at [company]
Responsibilities:
[paste your current bullet points or description]
```

### 3.4 — Salary Negotiation Script
```
I'm negotiating salary for a [job title] role. The offer is [amount].
I want [target amount]. My leverage:
- [your experience/skills]
- [market data if you have it]
- [competing offers if any]

Write me a confident, professional negotiation script I can use
on the call. Not aggressive. Not desperate. Just clear and prepared.
```

### 3.5 — Client Proposal
```
Write a project proposal for:
Client: [who they are]
Project: [what they need]
My approach: [how you'll do it]
Timeline: [estimated]
Budget: [range]

Keep it under 1 page. Professional but warm. Focus on their
outcome, not my process.
```

### 3.6 — Pricing Strategy
```
I sell [product/service] to [audience]. Currently charging [price].
My costs are approximately [costs]. Competitors charge [range].
Help me think through:
1. Am I undercharging or overcharging?
2. What pricing model would maximize revenue?
3. How should I present the price to reduce objections?
4. What would a premium tier look like?
```

## SECTION 4: Marketing & Growth

### 4.1 — Marketing Plan in 10 Minutes
```
Create a 30-day marketing plan for [business/product].
Budget: [amount or "zero — organic only"]
Target audience: [who]
Goal: [what you want to achieve — followers, sales, leads]
Channels available: [list what you use — social, email, etc.]
Give me specific daily/weekly actions, not vague advice.
```

### 4.2 — Ad Copy Generator
```
Write ad copy for [product/service].
Platform: [Facebook / Google / Instagram / LinkedIn]
Target audience: [who]
Main benefit: [what they get]
Offer: [discount, free trial, etc. — or none]
Write 3 versions: emotional, logical, and urgency-based.
Keep each under [character limit for platform].
```

### 4.3 — SEO Content Strategy
```
I want to rank on Google for topics related to [your niche].
My website is about [what you do].
Give me:
1. 10 keyword ideas I should target (mix of easy and ambitious)
2. Content type for each (blog post, guide, comparison, etc.)
3. Title suggestions optimized for search
4. Which 3 to write first based on difficulty vs. traffic potential
```

### 4.4 — Email Sequence Builder
```
Build a 5-email welcome sequence for new subscribers.
My business: [what you do]
What they signed up for: [lead magnet / newsletter]
Goal of the sequence: [build trust / sell product / educate]

For each email give me: subject line, preview text, body outline
(~150 words), and CTA. Space them [daily / every 2 days / weekly].
```

### 4.5 — Landing Page Copy
```
Write landing page copy for [product/service].
Target audience: [who]
Main pain point: [what problem you solve]
Key benefits: [list 3-4]
Social proof: [any numbers, testimonials, credentials]
CTA: [what you want them to do]

Structure: Headline → Subheadline → Problem → Solution → Benefits
→ Social Proof → CTA → FAQ
```

## SECTION 5: AI-Specific Prompts (Using AI to Use AI Better)

### 5.1 — Find the Right AI Tool for Your Task
```
I need to [describe what you're trying to do]. I'm not sure which
AI tool is best for this. I have a budget of [amount/month] or
prefer free tools.
Compare the top 3 options. For each: name, cost, what it's best at,
biggest limitation, and your recommendation for my situation.
```

### 5.2 — Build an AI Workflow
```
I do [this task] every [frequency] and it takes me [time].
Design an AI-powered workflow that automates or speeds this up.
Be specific: which tools, what order, what I still do manually.
Estimate how much time I'd save. Keep it realistic — I'm one person,
not a tech company.
```

### 5.3 — Prompt Engineering for Better Results
```
I keep getting mediocre results when I ask AI to [task].
Here's the kind of prompt I've been using: [paste your prompt].
What am I doing wrong? Rewrite my prompt to get much better results.
Explain what you changed and why.
```

### 5.4 — AI-Powered Research
```
Research [topic] for me. I need:
1. Current state of things (as of 2026)
2. Key players and their positions
3. Main trends and predictions
4. Controversies or risks
5. 3 sources I should read for deeper understanding

Be specific with names, numbers, and dates. No vague summaries.
```

### 5.5 — Create Custom GPT Instructions
```
I want to set up a custom AI assistant for [specific use case].
It should:
- Always respond in [tone/style]
- Know about [your industry/context]
- Follow these rules: [list any constraints]
- Format outputs as [how you want results]

Write the system instructions I should paste into Custom GPT /
Claude Project / Gems settings.
```

### 5.6 — Automate Your Week with AI
```
Here's how I spend my typical work week:
[describe your weekly tasks and time spent on each]

Identify every task that AI could help with. For each one:
- Which AI tool to use
- The exact prompt or workflow
- Estimated time saved
- Difficulty to set up (easy / medium / hard)

Prioritize by biggest time savings with easiest setup.
```

## SECTION 6: Personal Development

### 6.1 — Life Decision Framework
```
I'm facing a decision: [describe the choice].
I'm torn because [what's making it hard].
Walk me through this like a thoughtful advisor:
1. What am I actually optimizing for? (help me clarify)
2. What would I regret more — doing it or not doing it?
3. What's the reversible vs. irreversible part of this decision?
4. What would you do, and why?
```

### 6.2 — Weekly Review
```
Help me do a weekly review. Here's what happened this week:
Wins: [list accomplishments]
Struggles: [what was hard]
Unfinished: [what I didn't get to]

Help me:
1. Identify patterns (what's working, what's not)
2. Set 3 priorities for next week
3. Suggest one thing to stop doing
4. Suggest one thing to start doing
```

### 6.3 — Networking Message
```
I want to connect with [person's name/role] on [platform].
Context: [why you want to connect — shared interest, their work, etc.]
Goal: [what you hope to get — advice, collaboration, introduction]

Write a short, genuine message that:
- References something specific about them
- Explains why I'm reaching out
- Makes it easy for them to say yes
- Is NOT generic or salesy
Under 100 words.
```

## SECTION 8: Advanced — AI Power User

### 8.1 — Chain of Thought Problem Solving
```
I have a complex problem: [describe it].
Don't give me the answer immediately. Instead:
1. Break the problem into smaller parts
2. Analyze each part separately
3. Identify what information is missing
4. Consider 2-3 different approaches
5. Then give me your recommended solution with reasoning
Show your work.
```

### 8.2 — Build a Custom Automation
```
I want to automate this workflow:
Step 1: [what happens]
Step 2: [what happens next]
Step 3: [etc.]

Currently I do this [manually / with tool X].
Design the automation using [Make.com / Zapier / n8n / whatever].
Give me the exact trigger, actions, and connections.
Include error handling — what happens if a step fails?
```

### 8.3 — Data Analysis Without Code
```
I have this data: [paste a sample or describe the dataset].
I need to understand: [what question you want answered].
Analyze it and give me:
1. Key patterns or trends
2. Outliers or surprises
3. A clear answer to my question
4. A simple visualization suggestion
5. What additional data would make this analysis better
```

### 8.4 — System Prompt for Your Business
```
Create a comprehensive AI system prompt for my business:
Business: [what you do]
Target customer: [who you serve]
Brand voice: [how you sound]
Common tasks AI will help with: [list them]
Rules: [any do's and don'ts]

The system prompt should make the AI behave like a knowledgeable
employee at my company who understands our customers, our voice,
and our standards.
```

### 8.5 — Reverse Engineer What's Working
```
Analyze this [piece of content / landing page / ad / email] that's
performing really well:

[paste the content]

Tell me:
1. Why it works (specific techniques used)
2. The psychological triggers being used
3. The structure/framework behind it
4. How I can apply these techniques to my own [content type]
Give me a template I can reuse.
```
